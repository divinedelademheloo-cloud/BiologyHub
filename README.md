## 📖 About

**BiologyHub** is a single-page, browser-based Biology study hub designed for Senior High School (SHS) students in Ghana. It combines rich curriculum notes, interactive visualizations, document import (PDF/Word), rich-text editing, and export tools — all in one lightweight HTML file that runs entirely in your browser.

🌐 **Live Site:** [https://YOUR_USERNAME.github.io/BiologyHub/](https://YOUR_USERNAME.github.io/BiologyHub/)

## ✨ Features

- 📚 **Ghana SHS Curriculum-Aligned** — Structured Biology content built for local learners
- 📊 **Interactive Charts** — Powered by Chart.js for dynamic data visualizations
- 📄 **Document Import** — Upload and read `.docx` (via Mammoth.js) and `.pdf` files (via PDF.js)
- ✍️ **Rich-Text Editing** — Built-in Quill editor for taking and formatting notes
- 📥 **Export Anywhere** — Save your work as PDF (html2pdf.js) or Excel (SheetJS/xlsx)
- 💾 **Efficient Storage** — Uses LZ-String compression for fast local storage
- 🎨 **Modern UI** — Styled with Tailwind CSS for a clean, responsive look
- ⚡ **Zero Backend** — Pure client-side; no server or database required
- 📱 **Runs Anywhere** — Works on desktop, tablet, and mobile browsers
